# 메가커피 단체 주문 시스템 - 테스트 케이스 문서

> **작성일**: 2026-03-10
> **버전**: 1.0
> **테스트 도구**: Playwright
> **테스트 서버**: http://localhost:8000

---

## ⚠️ 중요 - 데이터 보호 정책

### DB 보호 원칙
- **모든 테스트는 테스트 데이터를 "Test{N}" 패턴으로 생성 후 테스트 완료 시 삭제**
- **절대로 기존 DB 데이터 삭제 금지**
- **테이블 전체 삭제 쿼리/로직 사용 금지**

### 테스트 데이터 명명 규칙
| 항목 | 명명 규칙 | 예시 |
|------|----------|------|
| 테스트 사용자 | TestUser{N}_{timestamp} | TestUser1_123456789 |
| 테스트 카테고리 | TestCategory{N} | TestCategory1 |
| 테스트 메뉴 | TestMenu{N} | TestMenu1 |
| 테스트 이력 | TestHistory{N} | TestHistory1 |

## 1. 메인 화면 (index.html) 테스트 케이스

### 1.1 화면 표시 기본 테스트

#### TC-MAIN-001: 페이지 로드 및 기본 레이아웃 확인
```

2. 페이지 로드 완료 대기 (networkidle)

[예상결과]
- 페이지 타이틀에 "메가커피" 포함
- 헤더에 "메가커피 단체 주문" 텍스트 표시
- 이름 선택 드롭다운(#nameSelect) 표시
```
